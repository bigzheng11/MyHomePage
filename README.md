# project-home-page

--用于个人主页

项目技术栈:`Vue3`+`TypeScript`+`Ant Design`


## 依赖
```
yarn install
```

## Ant Design
1. 安装
```
yarn add ant-design-vue@4.x --save
```

2.  引入 - 在 `main.ts` 中
```
import Antd from 'ant-design-vue'
import 'ant-design-vue/dist/reset.css';

createApp(App).use(Antd).mount("#app");
```

## Axios
1. 安装
```
yarn add axios
```
